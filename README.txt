# BaddBeatz Production & DJ Portfolio

A modern, responsive portfolio website for BaddBeatz Production featuring AI-driven music generation, content creation, and social media integration.

## Features

- 🎵 Music Generation (Techno, 138 BPM, Dark mood)
- 📱 Content Generation with Smart Recycling & A/B Testing
- 🔗 Social Media Integration
- 📊 User Dashboard
- 🌐 Multilingual Support
- 📱 Fully Responsive Design

## Technologies Used

- HTML5
- Tailwind CSS
- Google Fonts
- Font Awesome Icons

## Deployment Options

### Option 1: Netlify Drop (Quick Deploy)

1. Visit [Netlify Drop](https://app.netlify.com/drop)
2. Drag and drop the entire project folder
3. Your site will be instantly deployed with a unique URL
4. (Optional) Configure your custom domain in Netlify settings

### Option 2: GitHub Pages (Automated Deploy)

1. Create a new repository on GitHub
2. Push the code to your repository:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/baddbeatz-portfolio.git
git branch -M main
git push -u origin main
```
3. Go to repository Settings > Pages
4. Set source to "GitHub Actions"
5. Your site will be deployed automatically on every push to main branch

The site will be available at: `https://YOUR_USERNAME.github.io/baddbeatz-portfolio`

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/baddbeatz-portfolio.git
```

2. Open `index.html` in your browser or use a local server:
```bash
python -m http.server 8000
```

3. Visit `http://localhost:8000` in your browser
